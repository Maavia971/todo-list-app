# 📋 To-Do List App

A simple and interactive **To-Do List** web application built using **HTML, CSS, and JavaScript**.  
This app allows users to **add, complete, delete, and clear tasks** with **localStorage** support so tasks are saved even after page reload.

## 🔹 Features

- Add new tasks ✅  
- Mark tasks as completed ✅  
- Delete individual tasks ❌  
- Clear all tasks at once 🗑️  
- Tasks saved using **localStorage** 🔒  
- Responsive and clean UI with subtle animations  

## 🔹 Technologies Used

- HTML5  
- CSS3  
- JavaScript (DOM manipulation & localStorage)  

## 🔹 How to Use

1. Open `index.html` in your browser  
2. Type a task in the input field and click **Add**  
3. Click on the task to mark it as **completed**  
4. Click **Delete** to remove a single task  
5. Click **Clear All** to remove all tasks
